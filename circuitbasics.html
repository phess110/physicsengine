Resistors

Resistors in series will each have the same current \(i\). The equivalent resistor should have voltage drop \(V_{\mathrm{eq}} = V_1 +\ldots + V_n = iR_1 + \ldots + iR_n\). On the other hand, \(V_{\mathrm{eq}} = iR_{\mathrm{eq}},\) whence $$R_{\mathrm{eq}} = R_1 + \ldots + R_n.$$ For resistors in parallel, the voltage drop across each resistor is the same, the equivalent resistor should have current \(i_{\mathrm{eq}} = i_1+\ldots + i_n\), hence \(\frac{V}{R_{\mathrm{eq}}} = \frac{V}{R_1} + \ldots +\frac{V}{R_n}.\) Therefore, $$\frac{1}{R_{\mathrm{eq}}} = \frac{1}{R_1} + \ldots + \frac{1}{R_n}.$$

Capacitors

Capacitors in series will each have the same charge \(q\). The equivalent capacitor should have voltage drop \(V_{\mathrm{eq}} = V_1 +\ldots + V_n = \frac{q}{C_1} + \ldots + \frac{q}{C_n}\). On the other hand, \(V_{\mathrm{eq}} = \frac{q}{C_{\mathrm{eq}}},\) whence $$\frac{1}{C_{\mathrm{eq}}} = \frac{1}{C_1} + \ldots + \frac{1}{C_n}.$$ For capacitors in parallel, the voltage drop across each capacitor is the same, the equivalent capacitor should have charge \(q_{\mathrm{eq}} = q_1+\ldots + q_n\), hence $$C_{\mathrm{eq}} = C_1+\ldots + C_{n}.$$

Inductors

Inductors in series will each have the same \(\frac{\mathrm{d}i}{\mathrm{d}t}\). The equivalent inductor should have voltage drop \(V_{\mathrm{eq}} = V_1 +\ldots + V_n = \frac{\mathrm{d}i}{\mathrm{d}t} (L_1 + \ldots + L_n)\). On the other hand, \(V_{\mathrm{eq}} = L_{\mathrm{eq}} \frac{\mathrm{d}i}{\mathrm{d}t},\) whence $$L_{\mathrm{eq}} = L_1 + \ldots + L_n.$$ For inductors in parallel, by analogous reasoning to resistors, $$\frac{1}{L_{\mathrm{eq}}} = \frac{1}{L_1} + \ldots + \frac{1}{L_n}.$$

Multiloop Circuits

The emf is defined by \(\mathcal{E} = \frac{\mathrm{d}W}{\mathrm{d}q}\), where \(\mathrm{d}W\) is the amount of work the emf device must do on the charge \(\mathrm{d}q\) to move it through the circuit.

  • Loop Rule: The sum of changes in potential in a complete traversal of any loop is zero.
  • Resistance Rule: Moving through a resistor in the direction of the current yields \(-iR\); in the other direction it yields \(iR\).
  • EMF Rule: Moving through an ideal emf device in the direction of the emf arrow (recall this points from negative to positive terminal) yields a change in potential of \(+\mathcal{E}\); in the other direction it is \(-\mathcal{E}.\)
  • Junction Rule: The current into any junction equals the current out of that junction.
