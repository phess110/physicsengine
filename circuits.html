The RC Battery Circuit

By the loop rule, $$R \frac{\mathrm{d}q}{\mathrm{d}t} + \frac{q}{C} = E.$$Consider the integrating factor \(v(t) = \exp(\frac{t}{RC}).\) Note \((v(t)q(t))' = \frac{q(t)v(t)}{RC} + q'(t)v(t) = \frac{E}{R} v(t)\), whence $$v(t)q(t) = \frac{E}{R} \int v(t)\mathrm{d}t = ECv(t) + c_0.$$ After dividing through by \(v\), we obtain \(q(t) = EC + c_0e^{-t/RC}\). Since \(q(0) = 0\), we can write $$q(t) = EC\left(1-e^{-t/RC}\right).$$ \(\tau = RC\).

The RL Battery Circuit

By the loop rule, $$L \frac{\mathrm{d}i}{\mathrm{d}t} + R i = E.$$ By symmetry with the RC battery circuit equation, it follows that $$i(t) = \frac{E}{R}\left(1 - e^{-Rt/L}\right).$$ \(\tau = \frac{L}{R}.\)

The RLC Circuit

Suppose we have an RLC circuit with no external emf. By the loop rule $$L \frac{\mathrm{d}^2q}{\mathrm{d}t^2} + R \frac{\mathrm{d}q}{\mathrm{d}t} + \frac{q}{C} = 0.$$We can guess, \(q(t) = e^{\lambda t}\) and from the characteristic equation, we have: $$\lambda = -\alpha \pm \sqrt{\alpha^2 - \omega^2}$$where \(\alpha = \frac{R}{2L}\) and \(\omega = \frac{1}{\sqrt{LC}}\). The quantity \(\omega\) is called the natural angular frequency of the circuit.
Case 1. Underdamped (\(\omega > \alpha\)). In this case, \(\alpha^2-\omega^2 < 0\), so $$q(t) = e^{-\alpha t}(Q_0\cos(\sqrt{\omega^2-\alpha^2}t -\phi).$$
Case 2. Overdamped (\(\alpha > \omega\)). In this case, the discriminant is positive and has magnitude less than \(\alpha\), so both roots of the characteristic equation are negative reals. Hence $$q(t) = Ae^{\lambda_1 t} + Be^{\lambda_2 t}.$$
Case 3. Critically Damped (\(\alpha = \omega\)). In this case, the discriminant is zero, so $$q(t) = (A+Bt)e^{-\alpha t}.$$
In any case, the charge decays exponentially with time until most of the energy initially in the circuit has dissapated through the resistor.

The LC Circuit

By taking \(R = 0\) above we can solve for \(q\) in an LC circuit. Because we are in the underdamped case, $$q(t) = Q_0\cos(\omega t -\phi)$$ It follows that an LC circuit is a perfect oscillator.

The RLC Circuit with Alternating EMF

If we apply an external alternating emf of the form \(\mathcal{E} = E_m \sin(\omega_d t)\), the loop rule gives $$L \frac{\mathrm{d}^2q}{\mathrm{d}t^2} + R \frac{\mathrm{d}q}{\mathrm{d}t} + \frac{q}{C} = E_0 \sin(\omega_d t).$$ Equivalently, $$(\star) \quad L i'' + R i' + \frac{i}{C} = \omega_dE_m \cos(\omega_d t).$$We can guess a particular solution: \(i_p(t) = I\sin(\omega_d t -\phi)\). Plugging into \((\star)\), we obtain $$\begin{aligned} I\left(\frac{1}{C}-L\omega_d^2\right)\sin(\omega_d t-\phi) + RI\omega_d \cos(\omega_d t-\phi) &= \omega_d E_m\cos(\omega_dt+\phi-\phi)\\ &= \omega_d E_m \left(\cos\phi \cos(\omega_d t -\phi) - \sin\phi \sin(\omega_d t -\phi)\right). \end{aligned}$$Equating coefficients, we find $$I = \frac{E_m}{\sqrt{R^2 + (X_L-X_C)^2}}$$and $$\tan \phi = \frac{X_L - X_C}{R}$$where we recall that \(X_L = \omega_d L\) is the inductive reactance and \(X_C = \frac{1}{\omega_d C}\) is the capacitive reactance. Recall that these quantities are defined so that \(V_R = I R\), \(V_C= IX_C\), and \(V_L=IX_L\). Note: the quantity \(Z = \sqrt{R^2 + (X_L-X_C)^2}\) is called the impedance of the circuit.
Observe that if \(i\) is an arbitrary solution to \((\star)\), then $$L (i-i_p)'' + R (i-i_p)' + \frac{(i-i_p)}{C} = 0,$$i.e. \(i\) differs from \(i_p\) by a solution to the homogenous equation. Thus, $$i(t) = c_1e^{\lambda_1 t}+c_2e^{\lambda_2 t} + I \sin(\omega_d t - \phi),$$where

  • \(\lambda_1,\lambda_2 = \frac{-R}{2L} \pm \sqrt{\left(\frac{R}{2L}\right)^2 - \frac{1}{LC}}\)
  • \(I = \frac{E_m}{\sqrt{R^2 + (X_L-X_C)^2}}\)
  • \(\tan \phi = \frac{X_L - X_C}{R}\)

The transient currents \(c_1e^{\lambda_1 t}+c_2e^{\lambda_2 t}\) can be neglected after enough time passes. Note that the current amplitude, \(I\), is maximized when \(X_L = X_C\), i.e. \(\omega_d L = \frac{1}{\omega_d C}\) or \(\omega_d = \frac{1}{\sqrt{LC}} = \omega\). The condition \(\omega = \omega_d\) is called resonance.
The power consumed by an AC RCL circuit is \(P = i^2R = I^2R \sin^2(\omega_d t-\phi).\) The average value of \(\sin^2(at)\) over one period is \(\frac{1}{2}\), so $$P_{\mathrm{avg}} = \frac{1}{2}I^2R = \left(\frac{I}{\sqrt{2}}\right)^2 R.$$ We define \(I_{\mathrm{RMS}} = \frac{I}{\sqrt{2}}\), the root-mean-square current (similarly, for voltage and emf), so we can write \(P_{\mathrm{avg}} = I_{\mathrm{RMS}}^2 R = \frac{\mathcal{E}_{\mathrm{RMS}}}{Z} I_{\mathrm{RMS}} R =\mathcal{E}_{\mathrm{RMS}} I_{\mathrm{RMS}} \cos\phi\) since \(\cos\phi = \frac{R}{Z}.\) The value \(\cos\phi\) is called the power factor of the circuit.
